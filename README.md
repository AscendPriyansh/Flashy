## 📌 Flashy 
An interactive flashcard web app to practice and revise topics in a simple, distraction-free way.

### Overview
Flashy is a minimal and clean flashcard learning application that helps users revise concepts quickly using a question-and-answer format.

### What problem it solves
Many learners struggle with passive revision. Flashy makes revision active and interactive, helping users test themselves instead of just reading notes.

### Why I built it
I wanted to build a simple yet modern project using only HTML, CSS, and JavaScript while creating something actually useful for learning and revision.
